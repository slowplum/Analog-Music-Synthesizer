# Analog-Music-Synthesizer
A fully analog 8-note music synthesizer built entirely from discrete componentes with help of oscillators and op-amp based circuits. 
The system generates eight distinct notes spanning the C major scale(C4-C5), each produced by a dedicated astable multivibrator oscillator.
The project focuses on simplicity, modularity, and educational value while remaining practical enough to be implemented on a custom PCB

The complete design was simulated, converted into a PCB layout, verified using ERC and DRC checks, and prepared for fabrication.
The design was based on the fundamentals of analog circuits including oscillators, buffering, signal summation, amplification, PCB design and hardware implementation. 
Rather than relying on specialized synthesizer ICs, the design uses commonly available operational amplifiers, passive components, and discrete analog circuits, making it accessible for students interested in analog circuit design and electronic audio systems.
